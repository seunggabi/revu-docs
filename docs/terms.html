이용약관

최종 수정일: 2026년 2월 18일

제1조 (목적)

본 약관은 REVU Apply(이하 "서비스")의 이용과 관련하여 서비스 제공자와 이용자 간의 권리, 의무 및 책임사항을 규정함을 목적으로 합니다.

제2조 (서비스의 내용)

REVU Apply는 REVU 플랫폼에서의 캠페인 신청을 보조하는 Chrome 확장 프로그램입니다.

  • 캠페인 신청 메시지 자동 입력
  • 메시지 저장 및 관리
  • 단축키를 통한 빠른 신청

제3조 (이용 계약의 성립)

이용 계약은 이용자가 본 약관에 동의하고 서비스 이용 신청을 하면 성립됩니다. 로그인은 GitHub OAuth를 통해 이루어집니다.

제4조 (서비스 이용료)

  • 무료 플랜: 하루 10회 신청 제한
  • 프리미엄: GitHub Sponsors 후원 시 무제한 신청 (월 $3)
📌 안내

프리미엄 기능은 GitHub Sponsors를 통한 후원자에게 감사의 의미로 제공되는 혜택입니다. 상품 판매가 아닌 후원 보상입니다.

제5조 (후원 및 취소)

  • 후원은 GitHub Sponsors를 통해 진행됩니다.
  • 후원은 언제든 취소할 수 있으며, 취소 시 해당 월 말까지 프리미엄 기능을 이용할 수 있습니다.
  • 후원 취소는 GitHub Sponsors 대시보드에서 직접 처리할 수 있습니다.

제6조 (이용자의 의무)

  • 서비스를 부정한 목적으로 사용하지 않습니다.
  • 타인의 계정을 무단으로 사용하지 않습니다.
  • 서비스의 정상적인 운영을 방해하지 않습니다.

제7조 (서비스 제공자의 의무)

  • 안정적인 서비스 제공을 위해 노력합니다.
  • 이용자의 개인정보를 보호합니다.
  • 서비스 장애 발생 시 신속히 복구합니다.

제8조 (면책 조항)

  • REVU 플랫폼의 정책 변경으로 인한 서비스 이용 불가에 대해 책임지지 않습니다.
  • 이용자의 귀책사유로 인한 서비스 이용 장애에 대해 책임지지 않습니다.
  • 본 서비스는 REVU와 무관한 독립적인 서비스입니다.

제9조 (약관의 변경)

본 약관은 관련 법령 변경이나 서비스 정책 변경 시 수정될 수 있으며, 변경 시 서비스 내 공지합니다.

제10조 (분쟁 해결)

본 약관과 관련된 분쟁은 대한민국 법률에 따르며, 관할 법원은 서비스 제공자의 소재지 법원으로 합니다.
